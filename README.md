# BiliBiliStreamingLotteryMachine (Bilibili 直播抽奖姬) 🎟️

这是一个专为 Bilibili 主播设计的轻量级直播抽奖工具包。本目录包含两个完全独立运行的抽奖版本，均为绿色免安装程序，双击即可直接使用：

1. **Bilibili 直播互动长连版 (`lottery_app.exe`)** — 连接直播间，自动监听弹幕与礼物登记抽奖。
2. **独立主播手动版 / 离线版 (`manual_app.exe`)** — 完全离线，支持手动输入/管理名字进行快速抽奖。

## 1. 快速开始使用

无需配置 Python 或任何依赖环境，直接双击运行对应的 `.exe` 文件即可：

* **使用直播自动抽奖**：双击运行 **`lottery_app.exe`**，然后在浏览器打开：[http://127.0.0.1:5070/](http://127.0.0.1:5070/)
* **使用手动离线抽奖**：双击运行 **`manual_app.exe`**，然后在浏览器打开：[http://127.0.0.1:5080/](http://127.0.0.1:5080/)

## 2. Bilibili 直播互动长连版 (`lottery_app.exe`)

通过 Bilibili 开放平台官方长连接接口，实时获取直播间互动数据，公平公正地进行抽奖。

* **运行端口**：`5070`
* **功能特点**：
  * **弹幕自动登记**：可设定自定义过滤关键字（例如：发送“我要抽奖”自动登记）。
  * **礼物自动登记**：可限制特定礼物名称、赠送数量及最低电池价格（如：送出 1 个情书或价值 50 电池以上的礼物）。
  * **手动辅助登记**：在自动监听的同时，主播可在后台左侧输入框手动添加观众（如现场 VIP 或中途参与的特定用户）。手动登记的用户会带有专属的紫色 `手动` 徽章。
  * **去重登记规则**：支持开启/关闭重复登记。关闭后，同一个人在同一轮抽奖中仅占用一个名额，防止刷屏。
  * **抽奖与名单管理**：支持一键剔除特定中奖者、清空所有登记信息。
* **数据存储**：
  * 配置保存在本地 `lottery_config.json`。
  * 登记名单保存在本地 `entries.json`。

### 首次使用配置步骤：
1. 运行 `lottery_app.exe`。
2. 浏览器打开 `http://127.0.0.1:5070/`。
3. 点击 **「过滤规则配置」**：
   - 输入您的 **主播身份码 (idCode)**（可在 Bilibili 直播中心或开放平台获取）。
   - 根据需求勾选和配置弹幕关键字及礼物规则。
   - 点击 **「保存设置」**。
4. 返回主面板，点击 **「启动直播监听」** 按钮。当状态变为“正在监听”且日志区显示“长连鉴权成功”时，即表示已进入实时监控状态。

## 3. 独立主播手动版 (`manual_app.exe`)

适合离线活动、视频录制、手动输入特邀嘉宾名单等与直播间弹幕无关的抽奖场景。

* **运行端口**：`5080`
* **界面风格**：采用高质感的 **暗绿金 (Dark Emerald Green & Gold)** 配色设计，与紫色 B站版做视觉区分。
* **功能特点**：
  * 无需填写主播身份码，不依赖任何网络接口。
  * 极其简单的手动输入添加观众机制，支持快速名单搜索过滤。
  * 可以单独点击名字右侧的删除图标移除单人，或一键清空名单。
  * 同样支持防重复中奖逻辑，快速滚动动画抽取赢家。
* **数据存储**：
  * 名单数据保存在本地 `manual_entries.json`。

## 4. OBS 直播姬嵌入与兼容设计

很多主播喜欢将抽奖控制台嵌入到 **OBS Studio** 或 **Bilibili Link** 的「自定义浏览器源/自定义窗口」中进行操作。由于这些软件内置的 Webview 容器会**禁用或忽略浏览器的原生弹窗（如 `confirm` 或 `alert`）**，从而导致普通的“清空”或“删除”功能无法响应。

为了解决此兼容性问题，本工具特别做了如下优化：
1. **内联两步确认清空**：
   - 点击「清空登记」后，不会弹出浏览器确认对话框，而是将按钮变为红色**「再次点击确认」**并倒计时 3 秒。只有在 3 秒内再次点击才会执行清空，既防止误触，又完美兼容 OBS 内部窗口。
2. **防缓存加载**：
   - 所有的接口数据均带有严格的防缓存 HTTP 响应头，确保列表更新与清空后，OBS 容器内的数据能瞬间刷新，没有残留延迟。

## 5. 数据持久化与便携说明

* **全便携式设计**：
  - 本工具经过路径优化，所有的数据库文件（`entries.json`、`manual_entries.json`）和配置文件（`lottery_config.json`）都会自动创建在 **`.exe` 文件所在的同级目录中**。
  - 复制或打包整个文件夹到其他电脑，数据与配置依然会完整保留，可直接继续使用。
